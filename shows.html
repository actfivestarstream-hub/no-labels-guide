Glossary

用語集

音源 / Streaming

デジタル音源。ストリーミング（スミン）とダウンロード（ダウンロード）の両方を指す。音楽番組スコアで最も比重が大きい項目（50〜60%）。

スミン / Streaming

ストリーミング再生のこと。「スミン」とも呼ばれる。ファンダム用語として活動全体を指すこともある。

音盤 / Physical CD

フィジカルCD。Circle Chart（出荷ベース）またはHanteo Chart（販売ベース）でカウント。番組によって参照チャートが異なる。

初動 / First-week sales

リリースから最初の7日間の販売枚数。SHOW CHAMPIONはHanteo（First-week sales）を参照。初動が高いほど物販スコアに有利。

事前投票 / Pre-vote

放送前に行われる公式アプリ・サイトでの投票。番組によって有効期間や投票場所が異なる。

リアルタイム投票 / Live vote

放送中のみ有効な投票。M COUNTDOWN・Music Core・INKIGAYOで採用。放送時間に合わせて投票することが重要。

放送 / Broadcast

各局の音楽番組への出演・放送回数によるポイント。事務所側のブッキング次第で変動するため、ファンが直接コントロールしにくい項目。

Circle Chart

韓国の主要音楽チャート。国内（Domestic Digital）とグローバル（Global K-POP）の2種類があり、番組によって参照するチャートが異なる。

Hanteo Chart

Hanteo加盟店での実販売数をリアルタイム集計するチャート。SHOW CHAMPIONの物販スコアに使用。Weverse Shop・ユニバ・タワレコが対応店舗。

ダウンロード / Download

デジタル楽曲の購入（ダウンロード）。特にBugsはDLの比重が高い（DL80:スミン20）ため、ストリーミングよりDL購入の方がスコアへの貢献が大きい。

相対評価 / Relative Scoring

全番組で採用されているスコア方式。1位アーティストが満点を取り、他は相対的に計算される。ライバルアーティストの数字も重要。

二重カウント / Double count

INKIGAYOで採用。韓国国内のストリーミングは「グローバルチャート」と「国内チャート」の両方にカウントされるため、国内ストリーミングが特に有利になる現象。
